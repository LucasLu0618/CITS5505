# KidCode Hub — User Stories

**Project:** KidCode Hub
**Team:** Lucas, Sagar and Shee
**Unit:** CITS5505

---

| #  | Backlog Item (User Story) | Story Points |
|----|---------------------------|:------------:|
| 1  | As a **student**, I want to create an account with a username, email, and password, so that I can access KidCode Hub and track my learning progress. | 3 |
| 2  | As a **registered student**, I want to log in with my email/username and password, so that I can access my personalised dashboard and resume my coding journey. | 2 |
| 3  | As a **student**, I want to browse all available coding courses on the Courses page, so that I can find a course that matches my skill level and interests. | 2 |
| 4  | As a **student**, I want to click on a course and view its full details (description, difficulty, topics covered), so that I can decide whether to enrol in it. | 3 |
| 5  | As a **student**, I want to upload and submit my completed project, so that my teacher can review it and I can share my work with the community. | 5 |
| 6  | As a **student**, I want to browse the Student Project Showcase page, so that I can get inspired by other students' creative work and ideas. | 2 |
| 7  | As a **student**, I want the option to stay logged in between sessions using a "Remember me" checkbox, so that I don't have to re-enter my credentials every visit. | 1 |
| 8  | As a **student** who has forgotten their password, I want to use a "Forgot password?" link to reset my password via email, so that I can regain access to my account. | 3 |
| 9  | As a **teacher**, I want to view all project submissions from my students in one place, so that I can efficiently review and provide feedback on their work. | 4 |
| 10 | As a **teacher**, I want to create and publish a new coding course with a title, description, and difficulty level, so that my students have structured content to follow. | 5 |
| 11 | As an **administrator**, I want to view, edit, or remove registered user accounts, so that I can maintain a safe and appropriate learning environment for children. | 3 |
| 12 | As a **visitor**, I want to see a clear and welcoming homepage that explains what KidCode Hub offers, so that I can quickly understand the platform and decide to register. | 1 |
